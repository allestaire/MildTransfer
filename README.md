## Meld-CX Exam

![Demo](./demo.gif)

### Requirements

- Node >=20.9
- MySQL 8


### Start

- `cp .env.example .env`
- Replace the values
- `docker compose up -d`
- `npm install`
- `npx prisma migrate dev`
- `node ./prisma/seeders/index.js`
- `yarn dev` or `npm run dev`
